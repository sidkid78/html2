2. Volatility Harvesting & Dispersion Strategy

The "Super-Beta" Volatility Premium

MicroStrategy (MSTR) trades as a levered "super-beta" vehicle for Bitcoin due to its debt-funded BTC acquisitions and speculative equity options demand. This dynamics creates a persistent, structural premium in both realized volatility (RV) and implied volatility (IV) over spot BTC.

Strategy Mechanics

The strategy captures this volatility spread using a delta-neutral variance swap framework.

  • Spread Definition: Measures the 20-day annualized realized variance spread:
    Var_Spread = RV_MSTR,20^2 − RV_BTC,20^2
    Var_Spread=RV MSTR,20 2 ​ −RV BTC,20 2 ​
  • Signal Generation: A 60-day rolling Z-score is applied to the variance spread.
    Short Vol Dispersion: When Z > 1.5 Z>1.5, short MSTR variance and long BTC variance.
    Long Vol Dispersion: When Z < −1.5 Z<−1.5, long MSTR variance and short BTC variance.
  • Exit: Reversion of the Z-score to 0 0.

3. Cross-Asset Momentum & Lead-Lag Strategy

Informational Lead-Lag Effect: High-frequency analysis reveals that price discovery occurs first in highly liquid spot markets (BTC and ETH) before propagating to equity markets (MARA, RIOT, COIN). Spot BTC/ETH price momentum leads these crypto-exposed equities with a short-horizon latency of 1 to 5 minutes.

Predictive Signal & Implementation

The optimal leading momentum indicator is a 14-period Relative Strength Index (RSI) calculated on a 5-minute interval of spot BTC.

The core logic is implemented in Python as follows: 

import pandas as pd
import numpy as np

def run_backtest(btc_df, equity_df):
    # Align high-frequency datasets on timestamp
    df = pd.merge_asof(
        btc_df.sort_values('time'), 
        equity_df.sort_values('time'), 
        on='time', 
        suffixes=('_btc', '_eq')
    )
    
    # Calculate BTC RSI-14
    delta = df['close_btc'].diff()
    up = delta.clip(lower=0).rolling(14).mean()
    down = delta.clip(upper=0).abs().rolling(14).mean()
    rs = up / down
    df['btc_rsi'] = 100 - (100 / (1 + rs))
    
    # Signal Generation: Long if BTC RSI > 50, Short if < 50
    df['signal'] = np.where(df['btc_rsi'] > 50, 1, -1)
    
    # Calculate Strategy vs Benchmark returns (incorporating execution delay)
    df['strat_ret'] = df['signal'].shift(1) * df['close_eq'].pct_change()
    df['bench_ret'] = df['close_eq'].pct_change()
    
    return df[['strat_ret', 'bench_ret']].dropna().cumsum()
