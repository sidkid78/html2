Domain Report:
Biological Process Prediction

Mission Directive

Goal: Build a self-contained machine learning pipeline to predict Biological Process (BP) Gene Ontology terms for proteins based strictly on their amino acid sequences. You will use the provided training set of experimentally determined GO terms to train your model. Your ultimate objective is to optimize the maximum information-accretion weighted F1-measure specifically for the BP subontology across no-knowledge, limited-knowledge, and partial-knowledge protein targets. You do not need to consider Molecular Function or Cellular Component targets.

Executive Summary

This report synthesizes the development of a self-contained machine learning pipeline designed to predict Biological Process (BP) Gene Ontology (GO) terms from amino acid sequences. The pipeline integrates state-of-the-art protein language models with a specialized evaluation framework designed to optimize the maximum information-accretion weighted F1-measure (wFmax).

Key components include ESM-2 sequence embeddings, a hierarchical Information Content (IC) weighting system, a custom multi-label neural network architecture, and a knowledge-tiered evaluation module.

1. Information Content and Semantic Weighting

The foundation of the pipeline lies in its ability to handle the hierarchical and non-uniform nature of Gene Ontology terms. Worker-Biol-1 implemented a frequency-based calculation for Information Content (IC), defined as:

IC = -log10(p(t))

Crucially, the system implements hierarchical propagation, ensuring that the frequency of a specific term includes the counts of all its descendant terms in the GO graph. This methodology ensures that rare, highly specific biological processes are assigned higher weights than generic, high-level terms.

2. Sequence Feature Engineering via ESM-2

To bridge the gap between raw amino acid sequences and numerical machine learning inputs, the pipeline utilizes Meta AI’s ESM-2 (Evolutionary Scale Modeling) transformer model. Specifically, the facebook/esm2_t6_8M_UR50D model was selected to generate context-aware embeddings.

The feature extraction process transforms variable-length sequences into fixed-length 320-dimensional vectors using mean pooling across residue-level hidden states. This approach captures the biochemical and evolutionary context of the sequence while ensuring a uniform input space for the downstream classifier.

3. Model Architecture & Loss

The core predictive engine, developed by Worker-Biol-3, is a Multi-Layer Perceptron (MLP) specifically optimized for high-dimensional sequence embeddings.

  • BPClassifier Architecture: Incorporates BatchNorm1d and Dropout layers to mitigate overfitting given high dimensionality.
  • Information-Accretion Loss: Replaces standard BCE. Scales misclassification penalty based on Information Accretion (IA) weights of BP nodes.
  • Threshold Optimization: Iterates through dynamic decision thresholds to maximize wFmax and minimize Semantic Distance (Smin).

4. Multi-Tiered Evaluation Framework

To ensure the model is robust across different levels of biological understanding, Worker-Biol-4 implemented a validation framework that categorizes protein targets based on existing knowledge levels. This segmentation is critical for assessing generalization to "dark matter" proteins, mirroring CAFA competition challenges.

Tier 01: No-Knowledge

Targets with 0 existing annotations. Represents true "dark matter" proteins.

Tier 02: Limited-Knowledge

Targets with 1-3 existing annotations. Tests augmentation capabilities.

Tier 03: Partial-Knowledge

Targets with more than 3 existing annotations. Baseline performance metric.
